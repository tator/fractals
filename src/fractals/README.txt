This is a program to test and explore the mysteries of the Mandbrot set or Z(n+1) = Z(n)^d + C. 

Funtions:
  Altering the D variable:
    Up arrow:
       adds to the D variable the amount specified by the left and right arrows.
    Down arrow:  
      subtracts from the D variable the amount specified by the left and right arrows.
    Left arrow:
      divides the amount you can ajust the D value by ten.  
    Right arrow:
      multiplies the amount you can ajust the D value by ten.  
      
    After using the arrows to set the desiered D value, press the space bar to run the fractal of the D value.

  Zooming: 
    Left click:
      zoom in on spot by a factor of four.
    Right click:
      zoom out from spot by a factor of four.
    Middle click or Scroll wheel click:
      reset the view to the default view.
      
    Once zoomed in a few times, press S then right click to zoom out. The program will start to auto zoom out, to show the depth and complexity of this set.
